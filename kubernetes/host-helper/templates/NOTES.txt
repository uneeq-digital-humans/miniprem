NOTE: install this chart BEFORE (or alongside) digitalhuman-interface — the
kiosk image's nginx declares host-helper as an upstream and will not start
until the `host-helper` Service exists in the namespace.

On generic (non-appliance) clusters the audio/display/model-pull features
simply report unavailable — GPU and service health still work. No values
customization is required for a default install.
