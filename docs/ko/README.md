# 미니프렘 플랫폼

> LLM 통합, 실시간 얼굴 애니메이션, 모니터링 기능을 갖춘 종합적인 디지털 휴먼 플랫폼입니다.

## 개요

미니프렘은 디지털 휴먼 인터페이스(레니)와 LLM 기능(vLLM), 워크플로 자동화(플로우이즈), 종합 모니터링 툴(프로메테우스 + 그라파나)을 결합한 통합 플랫폼입니다. 이 설정을 사용하면 가상 휴먼 인터페이스를 통해 고급 AI 상호 작용을 배포하고 관리할 수 있습니다.

## 기능

- **디지털 휴먼 인터페이스**: 실시간 얼굴 애니메이션을 갖춘 Renny 기반
- LLM 통합**: 자연어 이해를 위한 Gemma3를 실행하는 vLLM
- 워크플로 자동화**: AI 워크플로우 구축 및 관리를 위한 Flowise
- **메트릭스 및 모니터링**: 실시간 성능 추적을 위한 Prometheus 및 Grafana
- **대기열 관리**: 안정적인 메시지 처리를 위한 Redis
- RIME AI**: 간단한 API를 통한 고품질 텍스트 음성 변환

## 빠른 시작

### 전제 조건

- 도커 및 도커 컴포즈
- 적절한 드라이버가 포함된 NVIDIA GPU
- 우분투 리눅스(권장)
- UneeQ의 필수 자격 증명(플랫폼 주소, API 키, 테넌트 ID)
- Azure Speech 서비스 자격 증명(지역 및 API 키)

### 설치

전체 설치 지침은 [시작 가이드](guides/getting-started.md)를 참조하세요.

## 플랫폼 구성 요소

MiniPrem에는 다음과 같은 구성 요소가 포함되어 있습니다:

- 레니**: 실시간 얼굴 애니메이션이 포함된 디지털 휴먼 인터페이스
- vLLM**: Gemma3와 함께 제공되는 대규모 언어 모델
- **Flowise**: AI 애플리케이션을 위한 시각적 워크플로 빌더
- 프로메테우스 및 그라파나**: 실시간 모니터링 및 대시보드
- Redis**: 안정적인 처리를 위한 메시지 큐
- **RIME**: 고품질 텍스트 음성 변환 엔진
