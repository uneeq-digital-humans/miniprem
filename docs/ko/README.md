# MiniPrem 플랫폼
LLM 통합, 실시간 얼굴 애니메이션 및 모니터링 기능을 갖춘 종합적인 디지털 휴먼 플랫폼.

## MiniPrem이란?

MiniPrem은 디지털 휴먼 인터페이스(Renny)와 강력한 언어 모델(vLLM),
워크플로우 자동화(Flowise) 및 모니터링 도구(Prometheus + Grafana)를 결합하여
가상 인간을 통한 생생한 AI 상호작용을 만들어냅니다.

## 주요 기능

- 🤖 **디지털 휴먼 인터페이스** - 실시간 얼굴 애니메이션
- 🧠 **LLM 통합** - Gemma3를 실행하는 vLLM
- 🔄 **워크플로우 자동화** - Flowise를 통한
- 📊 **모니터링** - Prometheus와 Grafana를 통한
