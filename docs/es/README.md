# Plataforma MiniPrem

Una plataforma humana digital integral con integración de LLM, animación facial en tiempo real y capacidades de monitoreo.

## ¿Qué es MiniPrem?

MiniPrem combina una interfaz humana digital (Renny) con potentes modelos de lenguaje (vLLM),
automatización de flujos de trabajo (Flowise) y herramientas de monitoreo (Prometheus + Grafana) para crear
interacciones de IA realistas a través de un humano virtual.

## Características Principales

- 🤖 **Interfaz Humana Digital** con animación facial en tiempo real
- 🧠 **Integración de LLM** con vLLM ejecutando Gemma3
- 🔄 **Automatización de Flujos de Trabajo** con Flowise
- 📊 **Monitoreo** con Prometheus y Grafana
