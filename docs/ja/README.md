# MiniPremプラットフォーム

> LLM統合、リアルタイム顔アニメーション、監視機能を備えた総合的なデジタルヒューマンプラットフォーム。

## 概要

MiniPremは、デジタルヒューマンインターフェース（Renny）とLLM機能（vLLM）、ワークフローオートメーション（Flowise）、および包括的な監視ツール（Prometheus + Grafana）を組み合わせた統合プラットフォームです。このセットアップにより、仮想ヒューマンインターフェースを介して高度なAIインタラクションを展開および管理できます。

## 特徴

- **デジタルヒューマンインターフェース**：Rennyを搭載し、リアルタイム顔アニメーションを実現
- **LLM統合**：自然言語理解のためのGemma3を実行するvLLM
- **ワークフローオートメーション**：AIワークフローの構築と管理のためのFlowise
- **メトリクス＆監視**：リアルタイムのパフォーマンス追跡のためのPrometheusとGrafana
- **キュー管理**：信頼性の高いメッセージ処理のためのRedis
- **RIME AI**：シンプルなAPIによる高品質のテキストツースピーチ

## クイックスタート

## プラットフォームコンポーネント

MiniPremには以下のコンポーネントが含まれています。

- **Renny**：リアルタイム顔アニメーションを備えたデジタルヒューマンインターフェース
- **vLLM**：Gemma3を使用した大規模言語モデルサービング
- **Flowise**：AIアプリケーション用のビジュアルワークフロー構築ツール
- **Prometheus＆Grafana**：リアルタイム監視とダッシュボード
- **Redis**：信頼性の高い処理のためのメッセージキュー
- **RIME**：高品質のテキストツースピーチエンジン
