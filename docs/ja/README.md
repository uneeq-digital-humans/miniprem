# MiniPremプラットフォーム

LLM統合、リアルタイムの顔のアニメーション、およびモニタリング機能を備えた包括的なデジタルヒューマンプラットフォーム。

## MiniPremとは？

MiniPremは、デジタルヒューマンインターフェース（Renny）と強力な言語モデル（vLLM）、
ワークフロー自動化（Flowise）、およびモニタリングツール（Prometheus + Grafana）を組み合わせて、
仮想人間を通じてリアルなAIインタラクションを実現します。

## 主な機能

- 🤖 **デジタルヒューマンインターフェース** - リアルタイムの顔のアニメーション
- 🧠 **LLM統合** - Gemma3を実行するvLLM
- 🔄 **ワークフロー自動化** - Flowiseによる
- 📊 **モニタリング** - PrometheusとGrafanaによる
