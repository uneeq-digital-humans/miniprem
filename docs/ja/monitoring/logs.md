# ライブコンテナログ

MiniPremスタックの各種サービスからのライブログを表示します。

## コンテナログ

ドロップダウンからサービスを選択してログを表示します:

```terminal
```container-logs
flowise
vllm
redis
prometheus
grafana
uneeq
```

## 動作原理

上記のターミナルは、各サービスのコントナログに接続します。これにより、次のことが可能になります:

1. リアルタイムに問題をデバッグする
2. アプリケーションのアクティビティを監視する
3. システムのパフォーマンスを追跡する

## ログ収集

ログはDockerのロギングシステムを使用して収集され、このインターフェースにストリーミングされます。本番環境では、次のようなより堅牢なロギングソリューションを検討することをお勧めします:

- ELKスタック（Elasticsearch、Logstash、Kibana）
- Loki（Grafanaスタックの一部）
- Datadogまたはその他のクラウド監視ソリューション
